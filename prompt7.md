小学5年生の漢字データ(193字)を整備してください。
これは子ども(小学5年生)が実際に使うアプリのデータなので、正確性が最重要です。

==========================================
1. 配当漢字リストの確定
==========================================
まず、文部科学省「小学校学習指導要領(平成29年告示)」の別表
「学年別漢字配当表」に基づく、現行(令和2年度施行以降)の
小学5年生の配当漢字 全193字 を確定してください。

【信頼できる出典の例】
- 文部科学省 学習指導要領
- 学習指導要領LOD (https://jp-cos.github.io/)
- Wikipedia「学年別漢字配当表」(現行版を参照、平成元年版は不可)

【重要な変更点】
平成29年告示で4年→5年に移った21字: 囲、紀、喜、救、型、航、告、殺、士、史、象、賞、貯、停、堂、得、毒、費、粉、脈、歴
平成29年告示で5年から外れた13字: 賀、群、徳、富、恩、券、承、舌、銭、退、敵、俵、預
これらが正しく反映された193字を採用してください。

【絶対NG】
旧版(平成元年版・185字)の5年生リストを使うこと。
特に「賀、群、徳、富、恩、券、承、舌、銭、退、敵、俵、預」が
5年生に含まれていたら、それは旧版なので必ず修正してください。

==========================================
2. 各漢字の問題データ作成
==========================================
193字それぞれについて、以下の条件で問題データを作成します。

【問題タイプの配分】
- 熟語(jukugo): 漢字が熟語の中で使われる問題。例「興奮(こうふん)」
- 送り仮名つき(okurigana): 動詞・形容詞など。例「営む(いとなむ)」
- 単漢字(single): 訓読みで意味が成立する字。例「桜(さくら)」
全体として、熟語:送り仮名:単漢字 = 4:3:3 程度を目安に
(漢字によって自然な形を優先。動詞性の漢字は送り仮名が自然、など)

【1問あたりのデータ形式】(既存JSONと同じ)
{
  "id": "g5-001",
  "grade": 5,
  "unit": "",   // 単元名は今回未対応(後で別タスク)
  "kanji": "興奮",
  "reading": "こうふん",
  "type": "jukugo",
  "example": "この本は{}する内容だ。",
  "meaning": "気持ちが高ぶること"
}

【例文の制約】(子どもが解くので超重要)
- 5年生までに習った漢字(1年〜5年配当の合計839字)とひらがな・カタカナのみで作る
- 6年生以降の漢字は使わない(子どもが読めない)
- 子どもの日常に近い、わかりやすい文脈
- 例文の {} の部分は、解答する漢字が入る場所
- 例: kanji="興奮" なら example="{}する場面が多い本だ。"

【意味の説明の制約】
- 5年生が読んでわかる平易な日本語
- 1〜2文程度

【1漢字あたりの問題数】
- 各漢字について、原則として1問だけ作る(193問)
- ただし、よく使われる漢字(例: 移、解、規、経 など)は2問作ってもOK

==========================================
3. 自己チェックを必ず実施
==========================================
データ作成が終わったら、以下のチェックを自分で行ってください。

(a) 漢字数チェック: 193字すべてが揃っているか、抜けや重複がないか
(b) 配当チェック: 5年生から外れた13字が混じっていないか
(c) 例文チェック: 各問の example に6年生以降の漢字が混じっていないか
   - 6年生配当漢字リスト(現行191字)を一時的に作って
     example の各漢字をチェック。混じっていたら例文を作り直す
(d) 読みチェック: kanji と reading が対応しているか
(e) ID チェック: g5-001 から g5-XXX まで連番になっているか

チェックの過程で発見した問題と、それをどう修正したかも報告してください。

==========================================
4. ファイルの出力
==========================================
- 出力先: public/data/kanji-grade5.json
- 既存ファイル(サンプル3問)は上書きしてください
- インデント2スペースで読みやすく整形

==========================================
5. 既存の3問のサンプルとの整合性
==========================================
既存の興奮(こうふん)、規則(きそく)、境界(きょうかい)は引き続き
データに含めて構いません。重複しないよう注意してください。

==========================================
変更点・問題点を以下の形式でレポート
==========================================
- 確定した193字のリスト(コンパクトに)
- 問題タイプの内訳(熟語○問、送り仮名○問、単漢字○問)
- 自己チェックで発見した問題と修正内容
- 完成した問題総数

長い作業になりますが、子どもの教育に使うデータなので、
時間がかかっても正確性を優先してください。
